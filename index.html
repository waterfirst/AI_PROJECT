🚀 AI & 데이터 분석 혁신 그룹

안녕하세요! 코딩, AI, 데이터 분석에 관심 있는 분들을 위한 흥미진진한 스터디 모임을 소개합니다.

📅 일시

  • 매주 일요일 오전 09:30 - 12:30 (3시간)
  • 시작일: 2024년 9월 8일 (추석 연휴 쉼)

📍 장소

스타벅스 (성복역 지점)

🎯 목표

최신 AI 기술을 활용한 실용적인 프로젝트 수행 및 개인 역량 강화 개인별 프로젝트 진행

📚 생각하고 있는 주요 주제

  1. 투자 분석 도구 개발
    • Pair Trading (KOSPI-S&P500, 원/달러-원/엔) 분석
    • R, Python을 활용한 코딩
    • R Shiny 또는 Streamlit을 이용한 웹 애플리케이션 개발
  2. AI 기반 특허 출원
    • AI를 활용한 특허 명세서 및 도면 작성
    • 대한민국 특허청에 직접 출원 프로세스 경험
  3. 화장품 산업 트렌드 분석
    • 로레알, 시세이도, 코스맥스 등 주요 기업의 10년간 연구 동향 분석
    • R의 메타 분석 패키지를 활용한 키워드 분석
    • 연구개발 트렌드 보고서 작성
  4. 과학/공학 웹 서비스 개발
    • R Shiny, Streamlit을 이용한 광학, 공학 관련 웹 서비스 구현
  5. AI API 활용 프로젝트
    • GPT, Claude, Gemini API를 이용한 혁신적인 웹 서비스 개발
  6. 최신 AI 도구 실습
    • Cursor IDE, 다양한 이미지 생성 AI 프로그램 학습 및 활용

🆕 추가 주제 제안

  1. 자연어 처리를 활용한 감성 분석 서비스
    • 소셜 미디어 데이터를 활용한 브랜드 평판 분석
    • BERT, RoBERTa 등 최신 NLP 모델 활용
  2. 컴퓨터 비전 프로젝트
    • 의료 영상 분석 (X-ray, MRI 등)을 통한 질병 진단 보조 시스템 개발
    • 농업 분야 작물 상태 모니터링 시스템
  3. 강화학습을 이용한 트레이딩 봇 개발
    • 가상화폐 또는 주식 시장 데이터를 활용한 AI 트레이딩 시스템 구현

💡 운영 방식

  1. 주간 프로젝트 로테이션
    • 매주 다른 주제로 미니 프로젝트 진행
    • 2-3주 단위로 심화 프로젝트 수행 가능
  2. 페어 프로그래밍
    • 2인 1조로 협업하여 문제 해결 능력 향상
  3. Lightning Talk
    • 매 모임 시작 시 15분간 최신 AI 기술 동향 공유
  4. 온라인 리소스 공유
    • GitHub 저장소를 통한 코드 및 자료 공유
    • Slack 채널을 통한 실시간 소통
  5. 분기별 해커톤
    • 1개월마다 팀 개인 프로젝트 발표 및 시연회 개최
